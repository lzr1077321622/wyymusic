# vue +vuex + 路由动画+路由守卫实现登录 ，

  用户登录跳转到登录页面 ，如果路由输入错误跳转到404页面。等用户登录过，再次访问登录页面，直接跳转到首页，在未登录的情况下，访问里面的路由跳转到登录页面，

```sh
npm install 安装依赖
```
```sh
npm run dev 启动项目
```
```sh
npm run build 打包项目
```
